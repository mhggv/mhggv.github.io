Ubuntu22.04.1和玩客云更换国内软件源详细教程

发表于2023-07-03|更新于2023-12-11|过程记录
|阅读量:

前言

本文主要是用来做备忘录的，尽可能详细的讲解Ubuntu和玩客云换源过程，再放上用到的工具下载链接以后再搞方便

为什么换原

Ubuntu中各种环境文件都要从国外一些站点去下载在国内可能无法访问，这里的源可以理解为下载的服务器端对你提供的下载地址，源表明了当下载或者更新某些软件的时候要从指定的源站点去下载或者更新，使用不同的源可能会影响我们的下载速度，这里我们介绍如何更换阿里云Ubuntu 镜像的源。

Ubuntu阿里云源

阿里云提供的有免费镜像源（同步更新官方源）我们找到需要的源地址：https://developer.aliyun.com/mirror/
找到Ubuntu，进去第一个就是，不过提供的例子没有22.04.1不懂的可能不会用下面就是

Ubuntu玩客云阿里云源

注意哈！玩客云是32位arm架构上面的不能用，不过阿里云提供的也有arm版本的ubuntu-ports的镜像

更换源

首先是修改源文件目录是/etc/apt/sources.list
修改方法有很多如果安装了可视化管理程序比如可道云，CasaOS等程序可以直接手动修改，把里面的源删除粘贴上面的源使用 sudo apt-get update 命令更新源
没有安装可视化管理程序的可以使用vim命令
vi /etc/apt/sources.list
进行编辑如果返回-bash: vi: command not found那就是没有安装vim编辑器使用sudo apt install vim命令安装，安装过程需要手动确定一下输入y即可。

  1. 打开文件 vi /etc/apt/sources.list
  2. 点击键盘 i 进行编辑不然无法编辑
  3. 把默认源全部删除，或者在每行开头添加#注释掉粘贴上面需要的源
  4. 按 Esc 建后输入: w （保存当前文件）: wq ( 保存并退出 vim）:wq!（强制保存并退出）: q! ( 不保存强制退出 vim)
  5. 使用下面两个命令更新源
    apt-get update 和 apt-get upgrade 是 Linux 系统中常用的命令，用于更新系统软件包列表和升级已安装的软件包
  6. apt-get update 命令会从软件源服务器下载最新的软件包列表，并将其存储在本地系统的缓存中。这样，当你需要安装或升级某个软件包时，系统就可以直接从本地缓存中获取该软件包的信息，而不需要再次从远程服务器下载。
  7. apt-get upgrade 命令则会检查本地缓存中的软件包版本信息，并尝试将所有已安装的软件包升级到最新版本。这包括安全补丁和其他新功能。如果有可用的更新会自动下载并安装这些更新。
